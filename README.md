# 💰 Money Manager – Frontend (React + Vite)


A modern, responsive **Personal Money Management Web Application** built using **React, Vite, and modern frontend practices**. This application helps users manage their income and expenses efficiently with a clean UI and secure integration with backend APIs.

---

## ✨ Features

* 🔐 **User Authentication** (Login & Sign Up)
* 📊 **Track Income & Expenses**
* 🧾 Categorized financial records
* 📈 Clear financial overview
* 🎨 Modern & responsive UI
* ⚡ Fast performance using Vite
* 🌐 Backend API integration
* ☁️ Deployed on **Vercel**

---

## 🖥️ Tech Stack

### Frontend

* ⚛️ React (Hooks based)
* ⚡ Vite

* 🌐 REST API Integration

## 🔐 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=your_backend_api_url
```

⚠️ **Note:**

* Variables must start with `VITE_`
* Restart the dev server after changes

## 🚀 Getting Started Locally

### 1️⃣ Clone the Repository

```bash
git clone [https://github.com/Dhiraj96k/MoneyManager-FE.git](https://github.com/Dhiraj96k/MoneyManager-FE.git)
```

### 2️⃣ Navigate to Project Folder

```bash
cd money-manager-fE
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```

---

## 🛠️ Build for Production

```bash
npm run build
```

## ☁️ Deployment

The application is deployed using **Vercel**.

🔗 **Live URL:** [https://moneymanagerfe.vercel.app/login](https://moneymanagerfe.vercel.app/login)
Steps used:

* Connected GitHub repository
* Added environment variables in Vercel dashboard
* Auto-deployment on every push to `main`
